#                                       HTML（结构）

### 简介

- 超文本标记语言
  - 超文本：超越了文本的限制，除了文字信息，还可以定义图片、音频和视频等
  - 标记语言：由标签构成的语言
- HTML运行在浏览器上，HTML标签由浏览器来解析
- HTML标签都是预定义好的，不区分大小写，属性值单双引号皆可



### 基础标签

|   标签    |           描述           |
| :-------: | :----------------------: |
| <h1>-<h2> |       定义标题大小       |
|  <font>   | 定义文本字体，尺寸，颜色 |
|    <b>    |       定义粗体文本       |
|    <i>    |       定义斜体文本       |
|    <u>    |      定义文本下划线      |
| <center>  |       定义文本居中       |
|    <p>    |         定义段落         |
|   <br>    |         定义折行         |
|   <hr>    |        定义水平线        |
|    <a>    |          超链接          |

### 表格标签

|  标签   |      描述      |
| :-----: | :------------: |
| <table> |    定义表格    |
|  <tr>   |     定义行     |
|  <td>   |   定义单元格   |
|  <th>   | 定义表头单元格 |

### 布局标签

|  标签  |    描述    |
| :----: | :--------: |
| <div>  |   占整行   |
| <span> | 占行内元素 |

### 表单标签

form的两个属性：

- action:指定表单数据提交的URL，想要被提交，必须在input中指定name属性
- method:指定表单的提交方式：
  - get：默认值，会拼接在URL后面
  - post：会在http请求协议中的请求体中
